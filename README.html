Publicar en GitHub Pages

  1. Crea un repo público llamado mapa-interactivo-turismo.
  2. Sube estos archivos (incluye mapa.png).
  3. En Settings → Pages, elige Branch: main y carpeta /(root).
  4. Tu URL quedará como https://TU-USUARIO.github.io/mapa-interactivo-turismo/.

Modo edición

Abre ?edit=1, selecciona un lugar y haz clic en el plano. Usa Exportar JSON y pega el contenido en data.json.

Datos de imagen

Si tu mapa.png tiene otro tamaño, edita data.json → imageWidth y imageHeight.
